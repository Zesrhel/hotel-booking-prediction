🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Issue: "git: command not found"
→ Install Git: https://git-scm.com/download/win

Issue: "Authentication failed"
→ Use GitHub Personal Access Token (not password)
→ Create at: https://github.com/settings/tokens

Issue: Repository not found
→ Check URL is correct and repository is PUBLIC

Issue: "Python not found"
→ Install Python 3.11+: https://python.org/downloads

Issue: Streamlit deployment stuck
→ Check GitHub Actions is complete
→ View logs at: github.com/YOUR_USERNAME/hotel-booking-prediction/actions

Issue: Model not loading
→ GitHub Actions must train model first (takes ~8 min)
→ Check Actions tab for training status
